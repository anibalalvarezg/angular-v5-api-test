# 🍺 Beer Management App

---

## ⚠️ Requisitos

**Node.js v12.22.12 (requerido)**

---

## 🚀 Inicio Rápido

```bash
# 1. Instalar dependencias
npm install

# 2. Iniciar servidor
npm start

# 3. Abrir navegador
http://localhost:4200
```

---

## 📋 Comandos

```bash
npm start              # Servidor desarrollo
npm run build          # Build producción
npm test               # Tests unitarios
npm run lint           # Verificar código
npm run format:write   # Formatear código
```

---
